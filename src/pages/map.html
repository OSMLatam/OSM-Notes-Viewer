Usage Examples

QGIS

  1. Open QGIS
  2. Go to Layer ‚Üí Add Layer ‚Üí Add WMS/WMTS Layer
  3. Click "New" and enter a name
  4. Paste the WMS URL above
  5. Click "Connect" and select the layer
  6. Click "Add"

ArcGIS

  1. Open ArcGIS
  2. Go to Insert ‚Üí Connections ‚Üí New WMS Server
  3. Enter the WMS URL
  4. Add the layer to your map
